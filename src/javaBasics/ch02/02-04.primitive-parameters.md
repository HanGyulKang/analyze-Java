# 기본형 매개변수

```java
class Data {
    int x;
}

class Main {
    public static void main(String[] args) {
        Data d = new Data();
        d.x = 10;
        System.out.println("main() : x = " + d.x);
        
        change(d.x);
        System.out.println("After change(d.x)");
        // 참조변수 d의 x값을 바꾼 것이 아니라 change method 안의 지역변수 x의 값만 바꿨기 때문에
        // 아래 결과값은 당연히 10이다.
        System.out.println("main() : x = " + d.x);
        
        // 진짜로 값을 바꾸고 싶다면 primitive type을 받는 메서드가 아닌
        // reference type의 객체를 받는 method를 만들어야한다.
        changeObject(d);
        System.out.println("main() : x = " + d.x); // 결과 >> main() : x = 1000
    }
    
    static void change(int x) { // 기본형 매개변수
        x = 1000;
        System.out.println("change() : x = " + x);
    }
    
    static void changeObject(Data data) {
        data.x = 1000;
    }
}

// [ result ]
// main() : x = 10
// change() : x = 1000
// After change(d.x)
// main() : x = 10
```
